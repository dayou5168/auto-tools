# Auto Tools

## 项目简介

Auto Tools 是一个用于在 Auto 网络（例如 Taurus 测试网）上执行账户操作和跨域转账的工具集合。
该项目旨在简化钱包管理、跨链交易以及与 Auto 网络的交互。

## 功能列表

- 钱包格式转换 //此功能马上废弃
- 跨域转账：支持向指定 EVM 地址发起跨域代币转账。



## 克隆仓库

``` bash
git clone https://github.com/dayou5168/auto-tools.git
cd auto-tools
```
